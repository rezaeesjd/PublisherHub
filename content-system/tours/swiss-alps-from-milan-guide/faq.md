# FAQ — Swiss Alps from Milan (Bernina route)

## Can I do the Swiss Alps from Milan in one day?
Yes. The Bernina + St Moritz route is commonly sold as a full-day trip (about 12–13 hours).

## Which stops are usually included?
Typical routing references Tirano, the Bernina scenic rail section, Alp Grüm/Diavolezza area, Pontresina, and St Moritz.

## Do I need a passport?
Yes. Source notes for the conversion tour state a valid passport is required.

## Is this route wheelchair accessible?
The source package for the conversion tour says it is not suitable for wheelchair users or guests with impaired mobility.
